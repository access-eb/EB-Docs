---
title: Add Exitbuilt team members to your WordPress site
---

For each of the following platforms, you will need to grant access to the following Exit Built employees:
- **neal@exitbuilt.com**  
- **danny@exitbuilt.com**  
- **mateo@exitbuilt.com**  
- **maxi@exitbuilt.com**  

1. Navigate to your WordPress site's dashboard.
2. In the **Users** tab, click the **Add New** button
    -or navigate by pasting this on your URL: yoursite.com**/wp-admin/users.php**
3. Add the Exit Built users with the appropriate role (*Administrator*).
    - don't forget to check the box that says "Send User Notification".
